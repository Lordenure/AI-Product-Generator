# TradeAI

> One product in. Full pack out.

If you've ever had to describe the same product over and over again for a site, a marketplace, ads, and social posts, this project is for that.

TradeAI takes one product and helps turn it into a full content pack. You give it the basics, and it starts shaping the rest: titles, benefits, descriptions, SEO copy, translations, FAQ, review replies, ad lines, and banner ideas.

The whole idea is simple. Less repeating yourself. Less jumping between tabs. Less time spent trying to rewrite the same thing in ten slightly different ways.

## Inside this project

- a frontend studio for creating and exploring product packs
- multilingual flows for different languages and selling platforms
- public and private pack spaces
- profile and account features with media uploads
- a .NET backend handling auth, users, and the app's core services

## Why it exists

TradeAI is not meant to replace taste, voice, or good ideas.

That is really the heart of it:
start with one product, get a strong first version of everything around it, and keep moving.

## Built with

Next.js, React, TypeScript, and ASP.NET Core.

Still growing, still being polished, and made with a lot of care.
